# The Dev Chest

A curated collection of Markdown notes including shortcuts, commands, scripts, and useful prompts for everyday development.

---

## About

This repository serves as a personal knowledge base where I store and organize:

- Frequently used commands
- Keyboard shortcuts
- Useful scripts
- Reusable prompts
- Notes and references

Everything is written in Markdown for simplicity, readability, and portability.
