Actúa como un desarrollador profesional con amplia experiencia en SQL y bases de datos relacionales (MySQL, PostgreSQL, SQL Server, Oracle).

Tu objetivo es recopilar y organizar las principales características y funciones de SQL en una estructura clara, completa y profesional.

### Requisitos:

1. Organiza las características en secciones lógicas basadas en buenas prácticas, por ejemplo:

   * Tipos de datos
   * Creación y modificación de tablas
   * Cláusulas básicas (SELECT, WHERE, ORDER BY, GROUP BY, HAVING)
   * Joins y relaciones entre tablas
   * Funciones agregadas (SUM, AVG, COUNT, MAX, MIN)
   * Funciones de texto (CONCAT, SUBSTRING, LENGTH, UPPER, LOWER)
   * Funciones de fecha y hora (NOW, DATEADD, DATEDIFF, EXTRACT)
   * Funciones numéricas (ROUND, CEIL, FLOOR, ABS)
   * Subconsultas
   * Operadores lógicos y de comparación
   * Índices y constraints
   * Transacciones y control de concurrencia
   * Procedimientos almacenados y triggers
   * Vistas y CTE (Common Table Expressions)
   * Utilidades avanzadas

2. Para cada sección:

   * Incluye una breve descripción de la categoría.
   * Crea una tabla con las funciones o características correspondientes.

3. Para cada función o característica:

   * Incluye una descripción clara de su propósito
   * Muestra la sintaxis exacta o un ejemplo mínimo de uso

4. Debes generar dos versiones completas del contenido:

   * Una en inglés
   * Una en español

5. Formato de las tablas:

   * En inglés:
     | Description | Function / Feature | Example |
   * En español:
     | Descripción | Función / Característica | Ejemplo |

6. Cada fila debe contener:

   * Una descripción clara y profesional
   * La función o característica exacta
   * Un ejemplo mínimo de uso cuando sea relevante

7. El resultado final debe:

   * Estar completamente consolidado en un único archivo en formato Markdown (.md)
   * Tener una estructura clara, profesional y fácil de navegar
   * No incluir ningún emoji ni símbolos innecesarios

8. Asegúrate de incluir funciones y características comunes y avanzadas de SQL, incluyendo:

   * SELECT, INSERT, UPDATE, DELETE
   * INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN
   * SUM, AVG, COUNT, MAX, MIN
   * CONCAT, SUBSTRING, LENGTH, UPPER, LOWER
   * NOW(), DATEADD(), DATEDIFF(), EXTRACT()
   * ROUND(), CEIL(), FLOOR(), ABS()
   * Transactions: COMMIT, ROLLBACK, SAVEPOINT
   * CREATE TABLE, ALTER TABLE, DROP TABLE
   * PRIMARY KEY, FOREIGN KEY, UNIQUE, CHECK, INDEX
   * CREATE VIEW, WITH (CTE)
   * Stored Procedures and Triggers syntax

9. Mantén consistencia en estilo, redacción y formato entre ambas versiones (inglés y español).
