# Markdown Shortcuts:

| Description                                      | Default key binding |
| ------------------------------------------------ | :-----------------: |
| Display all commands                             | `ctrl+M ctrl+M `    |
| Make **bold**                                    | `ctrl+B`            |
| Make _italic_                                    | `ctrl+I`            |
| Make [a hyperlink](www.example.org)              | `ctrl+L`            |
| Make an image ![](image_url.png)                 | `ctrl+shift+L`      |
| Make `a code block`                              | `ctrl+M ctrl+C `    |
| Make `inline code`                               | `ctrl+M ctrl+I`     |
| Make \* bullet point                             | `ctrl+M ctrl+B`     |
| Make 1. numbered list                            | `ctrl+M ctrl+1`     |
| Make - [ ] check list (Github flavored markdown) | `ctrl+M ctrl+X`     |
| Toggle # H1 title                                | -                   |
| Toggle ## H2 title                               | -                   |
| Toggle ### H3 title                              | -                   |
| Toggle #### H4 title                             | -                   |
| Toggle ##### H5 title                            | -                   |
| Toggle ###### H6 title                           | -                   |
| Add Tabular values                               | -                   |
| Add Tabular values with header                   | -                   |
