| Description                         |      Command      |
| ----------------------------------- | :---------------: |
| Colapsa todas las ventanas.         |     `Win + D`     |
| Historial de apps.                  |    `Win + Tab`    |
| Cambiar entre apps abiertas.        |    `Alt + Tab`    |
| Menú rápido.                        |     `Win + X`     |
| Bloquear el pc.                     |     `Win + L`     |
| Ir una página atras o adelante.     |  `Alt + izq/der`  |
| Abre el explorador de archivos.     |     `Win + E`     |
| Minimiza todas las ventanas.        |     `Win + M`     |
| .                                   | `Win + Shift + M` |
| Crea una carpeta nueva.             |   `Win + PrtSc`   |
| Limpia la consola.                  |     `Win + R`     |
| Mueve un archivo a otro directorio. |     `Win + S`     |

Windows:
. Win + d = Colapsa todas las ventanas.
. Win + Tab = Historial de apps.
. Alt + Tab = Cambiar entre apps abiertas.
. Win + x = Menú rápido.
. Win + l = Bloquear el pc.
. Alt + flecha atrás o adelante = Ir una página atrás o adelante.
. Win + e = Abre el explorador de archivos.
. Win + m = Minimiza todas las ventanas.
. Win + Shift + m = levanta las ventanas minimizadas.
. Win + PrtSc = Captura de ambas pantallas.
. Win + r = Run(Ejecutar).
. Win + s = Search.

File Explorer:
. Alt + d = Select directory.
. Shift + Tab = Se sitúa sobre el menú lateral.
