# VS Code Windows en ingles:

| Description                                                             | Command                        |
| ----------------------------------------------------------------------- | :----------------------------: |
| Abre y cierra la consola.                                               | `Ctrl + J / Ctrl + Ñ`          |
| Borra la linea y ajusta el texto                                        | `Shift + Del`                  |
| Navegar entre pestañas.                                                 | `Ctrl + PgUp / Ctrl + PgDn`    |
| Grabar todos los archivos.                                              | `Ctrl + K -> S`                |
| Posiciona un cursor al final de las lineas seleccionadas                | `\*\*Shift + Alt + I`          |
| Da formato al código.                                                   | `Shift + Alt + I`              |
| Comenta el texto seleccionado.                                          | `Ctrl + Shift + A`             |
| Da sugerencias.                                                         | `Ctrl + Space`                 |
| Selecciona la siguiente ocurrencia del caracter o palabra seleccionada. | `Ctrl + D`                     |
| para pintar palomas.                                                    | `Space`                        |
| Te sitúa en el explorador del proyecto.                                 | `Ctrl + Shift + E / Alt + 1`   |
| Cierra la pestaña actual.                                               | `Ctrl + W`                     |
| selecciona lineas completas.                                            | `Shift + arriba o abajo`       |
| Cierra y abre la ventana del explorador del proyecto.                   | `Ctrl + B`                     |
| mover linea.                                                            | `Alt + arriba/abajo`           |
| Sube o baja la pantalla.                                                | `Ctrl + up/down`               |
| copia linea up/down.                                                    | `Ctrl + Shift + Alt + up/down` |
| genera una archivo con formato.                                         | `Ctrl + Shift + p`             |
| abre el menu del botón derecho.                                         | `Fn + Shift + F10`             |
| Renderiza el archivo Markdown                                           | `Ctrl + Shift + V`             |
