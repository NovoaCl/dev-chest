# Directory nav

| Description                                 |                     Command                      |
| ------------------------------------------- | :----------------------------------------------: |
| Cambia el directorio a la carpeta indicada. |               `cd nombre-carpeta`                |
| Vuelve al directorio anterior.              |                      `cd..`                      |
| Lista las carpetas en el directorio actual. |                      `dir`                       |
| Crea una carpeta nueva.                     |              `mkdir nombre-carpeta`              |
| Limpia la consola.                          |                      `cls`                       |
| Mueve un archivo a otro directorio.         | `move "directorio-archivo" "directorio-destino"` |
