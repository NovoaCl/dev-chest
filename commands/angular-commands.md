## Angular Commands Table

| description                                    | command                                  |
| ---------------------------------------------- | ---------------------------------------- |
| Create a new Angular project                   | `ng new app-name`                        |
| Create project without installing dependencies | `ng new app-name --skip-install`         |
| Create project without routing                 | `ng new app-name --routing=false`        |
| Run development server                         | `ng serve`                               |
| Run server on specific port                    | `ng serve --port 4201`                   |
| Open browser automatically                     | `ng serve --open`                        |
| Generate component                             | `ng generate component name`             |
| Generate component (shortcut)                  | `ng g c name`                            |
| Generate service                               | `ng g service name`                      |
| Generate module                                | `ng g module name`                       |
| Generate module with routing                   | `ng g module name --routing`             |
| Generate guard                                 | `ng g guard name`                        |
| Generate interface                             | `ng g interface name`                    |
| Generate pipe                                  | `ng g pipe name`                         |
| Generate directive                             | `ng g directive name`                    |
| Generate class                                 | `ng g class name`                        |
| Build project                                  | `ng build`                               |
| Production build                               | `ng build --configuration production`    |
| Build with base href                           | `ng build --base-href /path/`            |
| Run unit tests                                 | `ng test`                                |
| Run tests headless                             | `ng test --watch=false`                  |
| Run e2e tests                                  | `ng e2e`                                 |
| Lint project                                   | `ng lint`                                |
| Show help                                      | `ng help`                                |
| Show Angular CLI version                       | `ng version`                             |
| Update Angular                                 | `ng update`                              |
| Update specific packages                       | `ng update @angular/core @angular/cli`   |
| Add library/package                            | `ng add package`                         |
| Remove package manually                        | `npm uninstall package`                  |
| Run custom architect target                    | `ng run project:target`                  |
| Analyze bundle                                 | `ng build --stats-json`                  |
| Create workspace without app                   | `ng new name --create-application=false` |
| Generate application in workspace              | `ng generate application name`           |
| Serve specific app                             | `ng serve app-name`                      |
| Configure environment                          | edit `angular.json`                      |
| Clean Angular cache                            | `ng cache clean`                         |
| Check cache info                               | `ng cache info`                          |
